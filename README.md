#  Sistema de Gerenciamento de Produtos em Python

##  Sobre o projeto

Este é um sistema de gerenciamento de produtos desenvolvido em Python com foco em prática de programação, organização de código e aplicação de conceitos de orientação a objetos.

O sistema permite cadastrar, listar, buscar, editar e excluir produtos, além de salvar e carregar dados em arquivo JSON.

---

##  Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Busca por nome
* Edição de produtos
* Exclusão de produtos
* Identificação do produto mais caro
* Persistência de dados com JSON

---

##  Tecnologias utilizadas

* Python 3
* Programação orientada a objetos (POO)
* Manipulação de arquivos (JSON)

---

##  Estrutura do projeto

```
.
├── main.py        # Menu principal
├── funcoes.py     # Regras de negócio
├── produto.py     # Classe Produto
├── produtos.json  # Base de dados
```

---

##  Como executar

1. Clone o repositório:

```
git clone https://github.com/SEU-USUARIO/sistema-produtos-python.git
```

2. Acesse a pasta:

```
cd sistema-produtos-python
```

3. Execute o sistema:

```
python main.py
```
